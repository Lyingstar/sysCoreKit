countW/Chinese

Under Google's review. Current version: 2.0

Google Docs™ default word count function doesn't correctly count Chinese characters. Not anymore. This add-on provides real-time bilingual counting for Google Docs™. Counts CJK characters and English words with consistent rules, and highlights selected text counts when you select content.

  • Accurate CJK + English counting rules
  • Selection-aware "current" counts
  • Lightweight UI for daily writing

Word count levels:

  1. Displays the word count for the current tab.
  2. Displays the total word count for all tabs.
  3. Displays the word count for each tab (including sub-tabs).

tabTOC

Under development.

Cross-tab Table of Contents for Google Docs™. Builds a structured TOC across multiple tabs, supports prefix removal, and keeps formatting clean for long documents.

  • Multi-tab scanning and structured output
